3 Counties Communication Test

The 3 Counties Communication Test is a vital event aimed at ensuring effective communication across the counties. This test helps in evaluating the use of TTU to achieve wide area coverage across the counties of Herefordshire, Shropshire and Worcestershire.

Date: Friday, 5th December 2025

Status: Provisional

The test involves the use of linked Talk Through Units (TTU) and a primary location in each county with a small team. Additionally, there is an opportunity to test Worcestershire's internal six district locations, which requires a larger team. Times for these tests are to be planned.
